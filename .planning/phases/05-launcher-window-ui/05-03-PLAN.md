  <files>src-tauri/tauri.conf.json</files>
  <action>
The launcher window is currently positioned at the OS default location (top-left on most Windows systems). LWND-01 requires it to be centered on the primary monitor.

Update `src-tauri/tauri.conf.json` launcher window entry to add `"center": true`:

```json
{
  "label": "launcher",
  "title": "Riftle",
  "width": 640,
  "height": 60,
  "center": true,
  "decorations": false,
  "transparent": false,
  "shadow": true,
  "resizable": false,
  "skipTaskbar": true,
  "alwaysOnTop": true,
  "visible": false,
  "focus": false
}
```

This ensures the launcher appears centered on the primary monitor when shown for the first time. Tauri's `center: true` positions the window at screen center on initial creation. Dynamic repositioning (e.g., on hotkey show) is handled in Phase 9 (Global Hotkey).

Verify the JSON is still valid after edit.

- Run `pnpm tauri dev`

**Step 3 — Keyboard navigation**
