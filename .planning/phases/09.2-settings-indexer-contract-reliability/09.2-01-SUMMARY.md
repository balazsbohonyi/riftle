## Decisions Made

- `test_get_settings_cmd_json_includes_allowlist_field` constructs the correct json!() shape (the "expected" output) rather than calling the Tauri command directly, since Tauri commands require AppHandle and cannot be unit tested. This documents the exact key the production code must include.
- `test_reindex_uses_live_settings` uses a real `tempdir()` path to bypass the `p.exists()` guard in `get_index_paths`, which filters out paths that don't exist on disk. Without a real path the assertion would never trigger.
- Timer interval tests use inline `if interval_mins == 0 { None } else { Some(...) }` expressions rather than calling production functions — this immediately documents the new conditional shape the timer loop must adopt in Plan 02.
